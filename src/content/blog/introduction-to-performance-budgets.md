Performance budgets guide decisions about which site features to include while maintaining optimal load times.

## Why Performance Budgets Matter

A site loading in under 3 seconds has higher completion rates than one taking 3.5 seconds. At 4 seconds, you could lose half your users.

## Key Guidelines

- **Mobile pages** should be approximately 20% lighter than desktop versions
- **Critical file size** should remain below 170 KB for optimal rendering
- **Above-the-fold content** requires faster CSS extraction and delivery
- Lighter pages with fewer requests and smaller byte sizes load quicker
- Mobile bandwidth and latency constraints require stricter asset budgets
- Budget allocation may shift between asset types based on page needs
